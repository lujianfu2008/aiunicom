## 本次问题

### 错误位置
在 `formatSmartContent` 函数中，代码块内容累积时使用了错误的转义：

```javascript
// ❌ 错误：双反斜杠
codeContent += line + '\\n';
```

### 修复内容
```javascript
// ✅ 正确：单反斜杠
codeContent += line + '\n';
```

## 转义规则（Python 字符串中的 JavaScript）

### 规则 1：普通字符串中的换行符
```python
# Python 字符串
js_code = """
    str.split('\n')    // ✅ JavaScript 收到：换行符
    str.split('\\n')   // ❌ JavaScript 收到：反斜杠+n
"""
```

### 规则 2：正则表达式中的换行符
```python
# Python 字符串
js_code = """
    str.replace(/\n/g, '<br>')    // ✅ JavaScript 收到：匹配换行符
    str.replace(/\\n/g, '<br>')   // ❌ JavaScript 收到：匹配反斜杠+n
"""
```

### 规则 3：字符串拼接中的换行符
```python
# Python 字符串
js_code = """
    code += line + '\n'    // ✅ JavaScript 收到：换行符
    code += line + '\\n'   // ❌ JavaScript 收到：反斜杠+n
"""
```
