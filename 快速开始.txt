========================================
    沃工单知识库系统 V3 - 快速开始
========================================

【启动方式】
推荐方式：双击 "启动系统.bat" 文件启动系统

备选方式：
1. 在命令行运行: python query_tool.py
2. 使用PowerShell运行: powershell -ExecutionPolicy Bypass -File run.ps1

【首次使用】
1. 系统启动后会自动检查环境（Python、Redis、依赖包）
2. 如果知识库未初始化，会提示使用 'init' 或 'rebuild' 命令
3. 输入 'init' 进行全量重建（首次使用推荐）
4. 等待初始化完成（根据文件数量可能需要几分钟）

【基本查询】
直接输入问题描述，例如：
  用户开户失败怎么办？
  这笔订单有什么问题？
  产品结束服务没有结束

【按类型查询】
使用 type: 前缀指定问题类型：
  type:开户报错 用户办理失败
  type:产品受理 服务未结束

【精确查询】
输入订单号或手机号：
  1325122268818964
  13800138000

【常用命令】
  help        - 显示帮助信息
  categories  - 显示问题分类
  stats       - 显示统计信息
  config      - 配置智普API Key
  init        - 全量重建（清空后重建）
  rebuild     - 增量更新（只处理新文件）
  quit/exit   - 退出系统

【注意事项】
1. 确保Redis Stack服务已启动
2. 确保数据目录存在: D:\工作文档\cbss2.0体验版\沃工单问题定位
3. 确保本地模型存在: D:\models\text2vec-base-chinese
4. 如需AI分析功能，请配置智普API Key

【故障排除】
1. 如果启动失败，运行 check_env.py 检查环境
2. 如果Redis连接失败，检查Redis服务是否启动
3. 如果查询无结果，使用 'init' 或 'rebuild' 初始化知识库
4. 如果双击启动文件无反应，请右键选择"以管理员身份运行"

【系统要求】
- Windows 10/11 或 Windows Server 2016+
- Python 3.8+
- Redis Stack 6.2+
- 内存：8GB及以上
- 硬盘：10GB可用空间

【技术支持】
如有问题请联系系统管理员
