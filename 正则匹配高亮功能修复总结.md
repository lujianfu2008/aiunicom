# 正则匹配高亮功能修复总结

## 问题说明

### 用户反馈
1. **只要是匹配正则的内容都应该高亮**，不只是手机号
2. **各模块应该独立**：内容查找、智能搜索、智能问答、正则查询互不影响

## 修复内容

### 1. 简化 `highlightRegexContent` 函数

**修改前**：有大量的特殊逻辑处理，只高亮手机号，不高亮其他匹配的内容 ❌

```javascript
function highlightRegexContent(text, pattern) {
    // 尝试直接使用原始正则
    try {
        const regex = new RegExp(pattern, 'gi');
        return escapedText.replace(regex, "<span class='highlight'>$&</span>");
    } catch (e) {
        console.warn('正则表达式无法直接应用:', e);
    }
    
    // ❌ 错误：各种特殊处理，导致很多匹配的内容不高亮
    if (pattern.includes('|')) {
        // 处理组合正则...
    }
    
    if (pattern.includes('\\d')) {
        // 只高亮手机号，其他数字不高亮
        if (pattern.includes('1[3-9]')) {
            const phoneRegex = /1[3-9]\d{9}/g;
            return escapedText.replace(phoneRegex, "<span class='highlight'>$&</span>");
        }
        return escapedText; // 其他包含\d 的模式不高亮
    }
    
    // ... 更多特殊处理
}
```

**修改后**：直接使用正则表达式高亮所有匹配的内容 ✅

```javascript
// 高亮正则匹配的内容 - 用于正则查询
function highlightRegexContent(text, pattern) {
    try {
        // 先转义 HTML 特殊字符
        const escapedText = escapeHtml(text);
        
        // 直接使用正则表达式匹配并高亮所有匹配的内容
        try {
            const regex = new RegExp(pattern, 'gi');
            return escapedText.replace(regex, "<span class='highlight'>$&</span>");
        } catch (e) {
            console.warn('正则表达式无法直接应用:', e);
            return escapedText;
        }
        
    } catch (e) {
        console.error('高亮错误:', e);
        return escapeHtml(text);
    }
}
```

### 2. 保持各模块独立

**内容查找**：
- 使用 `highlightAndEscape()` 函数
- 只高亮搜索关键词
- 不受正则查询影响

**智能搜索**：
- 使用 `highlightAndEscapeExact()` 函数
- 只高亮搜索关键词（精准匹配）
- 不受正则查询影响

**智能问答**：
- 使用 `highlightAndEscape()` 函数
- 只高亮问题关键词
- 不受正则查询影响

**正则查询**：
- 使用 `highlightRegexContent()` 函数
- 高亮所有匹配正则的内容
- 不影响其他模块
