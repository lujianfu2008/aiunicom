# 系统信息面板优化说明

## 修改内容

### 1. 移除系统信息面板中的"知识库管理"部分
- **位置**: 系统信息标签页
- **移除内容**: "知识库管理"卡片及其"查看所有文档"按钮
- **原因**: 该功能没有实际实现，且知识库管理接口已在 API 文档中说明

### 2. 保留的功能
系统信息面板现在包含以下三个部分：
1. **问题分类** - 显示所有问题类型分类
2. **统计信息** - 显示文档总数、分类数量、自动监控状态等
3. **配置信息** - 显示和编辑系统配置

### 3. 知识库管理接口说明
知识库管理的相关接口已经在 API 文档标签页中详细说明：

#### POST /api/init
- **功能**: 全量重建知识库（清空所有数据）
- **使用场景**: 需要完全重建知识库时使用

#### POST /api/rebuild  
- **功能**: 增量更新知识库（只处理变更文件）
- **使用场景**: 日常更新知识库时使用

## 修改的文件
- `api_server.py` - 移除了系统信息面板中的知识库管理 UI 部分

## 影响
- 系统信息面板更简洁清晰
- 用户需要查看知识库管理接口时，可以查看 API 文档标签页
- 不影响任何实际功能
