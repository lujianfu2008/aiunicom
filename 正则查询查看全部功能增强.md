# 正则查询查看全部功能增强

## 修复内容

### 1. 移除"打开文件"按钮

**问题**：正则查询结果列表中，每个项目旁边都有"打开文件"按钮，点击会出现错误。

**解决方案**：
- 移除了正则查询结果中的"打开文件"按钮
- 只保留"查看全部"按钮
- 文件标题不再可点击

**修改位置**：
```javascript
// api_server.py 第 1402-1415 行
```

### 2. 添加正则高亮支持

**问题**：正则查询的"查看全部"不显示高亮效果。

**解决方案**：
- 新增 `viewFileWithRegex` 函数，专门处理正则查询的查看全文
- 传递正则 pattern 参数到后端 API
- 使用 `highlightRegexContent` 函数高亮匹配的内容
- 支持智能格式、分段显示等多种展示方式

**修改位置**：
```javascript
// api_server.py 第 981-1048 行
function viewFileWithRegex(filePath, fileName, problemType, problemId, pattern) {
    // ...
    // 传递正则 pattern 参数
    fetch(`/api/view_file?file_path=${filePath}&file_name=${fileName}&pattern=${encodeURIComponent(pattern)}`)
        .then(r => r.json())
        .then(data => {
            const content = data.content || '无内容';
            
            // 使用正则高亮
            const highlightedContent = highlightRegexContent(content, decodedPattern);
            
            // 智能格式化
            const smartFormatted = formatSmartly(content, highlightedContent);
            
            window.formattedContent = {
                raw: `<pre>${escapeHtml(content)}</pre>`,
                smart: smartFormatted,
                section: sectionFormatted
            };
            
            document.getElementById('modal-content').innerHTML = toolbar + smartFormatted;
        });
}
```

## 高亮效果

### 正则查询高亮

当使用正则表达式 `1[3-9]\d{9}` 搜索手机号时：

**原文**：
```
用户反馈：联系电话 18692310000，备用电话 13912345678
时间是 2021-04-20 15:30:45
```

**高亮后**（智能格式）：
```
用户反馈：联系电话 <span class="highlight">18692310000</span>，
备用电话 <span class="highlight">13912345678</span>
时间是 2021-04-20 15:30:45（不高亮）
```

### 高亮特点

1. ✅ **精确匹配**：只高亮符合正则的部分
2. ✅ **不会误高亮**：日期时间中的数字不会被高亮
3. ✅ **多格式支持**：可在原始文本、智能格式、分段显示间切换
4. ✅ **所有运营商**：支持联通、移动、电信所有号段
