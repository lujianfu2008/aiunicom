## 修复原则

### 统一使用模板字符串

所有包含变量的 HTML 字符串，统一使用模板字符串：

```javascript
// ✅ 统一标准
element.innerHTML = `<div class="${className}">${content}</div>`;

// ❌ 禁止使用
element.innerHTML = '<div class="' + className + '">' + content + '</div>';
```

### 优点

1. **安全性**：避免特殊字符导致的语法错误
2. **可读性**：代码更清晰，易于维护
3. **一致性**：统一编码风格
